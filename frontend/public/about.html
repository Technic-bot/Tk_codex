About this Codex

Hello folks! I see you have landed in my tiny codex. This is intended as a way to quickly search for art and the contents of the comic. Pretty much like he excellent Tk Codex by Amenon

This is by no means intends to replace Amenon work, functionality is not as extensive as his site, but this is an open source project meaning anyone other than me can host it, it also has some features that allow it to be automatically up to date. You can check the project https://github.com/Technic-bot/Tk_codex for cloning and other details.
